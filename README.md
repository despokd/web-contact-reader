# Web contact reader
 Display VCF-file contact data readable in a website
## Features
- Upload vcf file with drag and drop
- Show instantly data
- use table view, with pagination, sorting, searching, details on click
- Use card view

## Used libraries
- [Vuesax 3](https://vuesax.com/)
- [Vue 3](https://v3.vuejs.org/)
